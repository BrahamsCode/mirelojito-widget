# 🕐 MiRelojito

Widget de reloj de escritorio hecho con **Electron**.

## Requisitos

- [Node.js](https://nodejs.org) v18 o superior

## Instalación

```bash
cd resources/app
npm install
```

## Ejecutar en desarrollo

```bash
npm start
```

## Generar el ejecutable (.exe)

```bash
npm run build
```

El `.exe` se genera en `resources/app/dist/`.

## Stack

- Electron
- HTML / CSS / JavaScript
